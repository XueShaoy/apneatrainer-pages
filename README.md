# GitHub Pages 仓库

这个仓库用于托管闭气+应用的静态页面。

## 页面列表

- [隐私政策](./apneatrainer-privacy-policy.html)

## 使用说明

这个仓库通过 GitHub Pages 自动部署。访问地址为：
`https://[你的用户名].github.io/pages/`

## 文件说明

- `apneatrainer-privacy-policy.html` - 闭气+应用的隐私政策页面
- `index.html` - 主页导航页面

## 更新说明

直接提交更改到 `main` 分支，GitHub Pages 会自动部署更新。
